# Szybka aktualizacja DartLiga PWA 1.3.0

1. Przed aktualizacją w aplikacji wybierz **Ustawienia → Eksportuj JSON**.
2. W repozytorium otwórz **Code → Add file → Upload files**.
3. Wgraj wszystkie pliki z paczki aktualizacyjnej do głównego katalogu repozytorium.
4. Zatwierdź nadpisanie i kliknij **Commit changes**.
5. W zakładce **Actions** poczekaj na zielony status workflow.
6. Otwórz stronę ponownie i sprawdź numer wersji **1.3.0**.
7. Gdy nadal widać poprzednią wersję, użyj `Ctrl + F5` albo zamknij i ponownie uruchom zainstalowaną PWA.

Folder `.github` pozostaje bez zmian.
